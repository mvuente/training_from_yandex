Для строительство двухмерной пирамиды используются прямоугольные блоки, каждый из которых характеризуется шириной и высотой.
Можно поставить один блок на другой, только если ширина верхнего блока строго меньше ширины нижнего. Самым нижним в пирамиде
может быть блок любой ширины. По заданному набору блоков определите, пирамиду какой наибольшей высоты можно построить из них.
Формат ввода
В первой строке входных данных задается число
N  — количество блоков (1 - 100000)

В следующих N  строках задаются пары натуральных чисел
w - ширина
h = высота (1 - 1000000000)

Формат вывода
Выведите одно целое число — максимальную высоту пирамиды.
